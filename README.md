# E-Commerce Website

## Description
A modern e-commerce website built with HTML and Tailwind CSS. This project was created as practice for internship preparation and demonstrates front-end web development skills.

## Technologies Used
- **HTML5** - For website structure and semantics
- **Tailwind CSS** - For styling and responsive design

## Features
- Responsive navigation bar
- Product display sections
- Modern footer design
- Mobile-friendly layout
- Clean and professional UI

## Project Structure
- Navbar with navigation links
- Multiple product sections
- Footer with additional information
- Responsive design for all devices

## What I Learned
- Implementing Tailwind CSS utilities
- Creating responsive layouts
- Git version control
- GitHub repository management

## How to View
1. Download or clone the project
2. Open `index.html` in any web browser
3. No additional setup required

## Future Improvements
- Add JavaScript functionality
- Implement shopping cart
- Add product filtering
- Connect to backend API
